# Grocery Speed Run Any%

## Description

Grocery Speed Run Any% is a Chrome extension that transforms your Amazon Fresh grocery shopping into an exciting speedrunning game. Challenge yourself to complete your grocery list as quickly as possible, adding a thrilling twist to your online shopping experience.

## Features

- Cyberpunk-inspired UI for a futuristic shopping experience
- Real-time stopwatch with millisecond precision
- Progress bar to visualize your shopping journey
- "Next" and "On Deck" items display for strategic planning
- Confetti celebration upon completion of your speed run

## How It Works

1. Install the Chrome extension
2. Navigate to Amazon Fresh
3. Find the Grocery Speed Run UI in the bottom right corner of your browser
4. Enter your grocery list in the provided text area
5. Click "Start Speed Run" to begin your shopping adventure
6. For each item:
   - The extension will automatically search for the item on Amazon Fresh
   - Click "Added" when you've added the item to your cart
   - Click "Skip" if you want to move to the next item without adding it
7. Once all items are processed, your final time will be displayed
8. Enjoy the confetti celebration of your grocery shopping victory!

## Installation

1. Clone this repository or download the ZIP file
2. Open Google Chrome and navigate to `chrome://extensions`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the directory containing the extension files
5. The Grocery Speed Run extension should now appear in your Chrome toolbar

## Usage

1. Go to Amazon Fresh (https://www.amazon.com/alm/storefront)
2. Look for the Grocery Speed Run UI in the bottom right corner of your browser
3. Enter your grocery list in the text area, with each item on a new line
4. Click "Start Speed Run" to begin your shopping adventure
5. Follow the on-screen instructions to add items to your cart
6. Complete your list as quickly as possible to set a new personal best!
